# Orchard Defense: Pixel Siege

เกม Tower Defense แบบพิกเซลสำหรับเว็บ เล่นได้ทั้งมือถือและคอมพิวเตอร์ โดยไม่ต้องติดตั้งไลบรารีเพิ่ม

## วิธีเล่น
1. เปิด `index.html`
2. กดเริ่มเกม
3. เลือกการ์ดยูนิตด้านล่าง แล้วแตะช่องบนสนาม
4. ป้องกันปราสาทให้ครบ 8 เวฟ
5. แตะยูนิตที่วางแล้วเพื่อขายคืน 40%

## ยูนิต
- Sunny: สร้างพลังงาน
- Peachy: ยิงระยะไกล
- Cobsworth: ยิงกระสุนคู่
- Spud: พลังชีวิตสูง ใช้ป้องกันแนวหน้า

## อัปขึ้น GitHub Pages
1. สร้าง Repository ใหม่
2. อัปโหลด `index.html`, `style.css`, `game.js`, `README.md`
3. ไปที่ Settings > Pages
4. เลือก Deploy from a branch และ branch `main` / root

## โครงสร้าง
- `index.html` หน้าเกม
- `style.css` หน้าตาและรองรับมือถือ
- `game.js` ระบบวางยูนิต ศัตรู เวฟ การยิง พลังงาน และเงื่อนไขชนะ/แพ้

สร้างเป็น MVP เพื่อใช้พัฒนาต่อ เช่น เพิ่มด่าน ตัวละคร บอส เสียง ร้านค้า ระบบบันทึก และสไปรต์ PNG จริง
