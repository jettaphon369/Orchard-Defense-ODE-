# Orchard Defense: Remastered

เกม Tower Defense สำหรับเว็บ รองรับมือถือและคอมพิวเตอร์

## เปิดเล่น
เปิดไฟล์ `index.html` หรืออัปโหลดไฟล์ทั้งหมดขึ้น GitHub Pages

## ระบบในเวอร์ชันนี้
- ตัวละคร 5 แบบ รวมยูนิตระเบิดทั้งแถว
- ศัตรูธรรมดา ศัตรูตัวใหญ่ และบอสเวฟ 8
- ระบบพลังงาน คูลดาวน์ แถบเลือด และจำนวนกำจัด
- เอฟเฟกต์กระสุน ระเบิด การสั่นหน้าจอ และข้อความกลางจอ
- UI ปรับใหม่สำหรับมือถือ
