# Orchard Defense Ultimate V5

เกมป้องกันสวนลอยฟ้าสำหรับเว็บและมือถือ สร้างด้วย Phaser 3

## ระบบใหม่ V5
- 12 เวฟ และบอสในเวฟ 4, 8, 12
- แถบเลือดบอสกลางหน้าจอ
- สกิลฮีโร่พายุสายฟ้า พร้อมคูลดาวน์
- เสียงเอฟเฟกต์แบบ Web Audio และปุ่มเปิด/ปิดเสียง
- แสดงจำนวนศัตรูที่กำจัด
- บันทึกสถิติสูงสุดด้วย Local Storage
- ศัตรูแข็งแกร่งขึ้นตามเวฟ
- แอนิเมชันโจมตี โดนโจมตี ตาย เอฟเฟกต์กระสุน ระเบิด และกล้องสั่น

## อัปโหลด GitHub Pages
นำไฟล์ทั้งหมดในโฟลเดอร์นี้ไปไว้ที่หน้าแรกของ Repository แล้วเปิด Pages จาก branch `main` และโฟลเดอร์ `/root`

เกมต้องเชื่อมต่ออินเทอร์เน็ตเพื่อโหลด Phaser จาก CDN
