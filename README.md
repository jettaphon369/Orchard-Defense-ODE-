# Orchard Defense: Sky Kingdom Hub V1

เวอร์ชันนี้เพิ่มเมืองลอยฟ้าแบบ Hub World ที่ใช้งานได้จริง และเชื่อมกับแคมเปญ 3 ด่านเดิม

## ระบบใหม่
- ตัวละครเดินไปยังอาคารเมื่อแตะ
- ร้านค้า เควส โรงตีเหล็ก ศาลาฮีโร่ ศูนย์รางวัล บ้านผู้เล่น และปราสาท
- ประตูผจญภัยเปิดหน้าเลือก 3 ด่าน
- ระบบ Gold, Gem, รางวัลรายวัน, เควส และอัปเกรดทดลอง
- บันทึกข้อมูลด้วย localStorage
- รองรับมือถือและ GitHub Pages

เปิด `index.html` ผ่านเว็บเซิร์ฟเวอร์หรืออัปโหลดไฟล์ทั้งหมดขึ้น GitHub Pages
