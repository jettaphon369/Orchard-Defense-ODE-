# Orchard Defense Ultimate V3

เกม Tower Defense สำหรับเว็บ รองรับมือถือและ GitHub Pages

## วิธีใช้งาน
1. อัปโหลด `index.html`, `style.css`, `game.js` ไปยัง root ของ Repository
2. เปิด Settings > Pages
3. เลือก Deploy from a branch และ branch `main` / root
4. รอ GitHub Pages อัปเดต

## ระบบใน V3
- 8 ยูนิต พร้อมบทบาทแตกต่างกัน
- 10 เวฟและบอสมังกรเงา
- ศัตรู Bug, Brute, Mage, Boss
- กระสุนปกติ, ยิงคู่, น้ำแข็ง, ปืนใหญ่, เลเซอร์ทะลุ
- ปุ่มความเร็ว x1/x2
- แถบ HP, คูลดาวน์, พลังงาน, การขายยูนิต
- รองรับหน้าจอมือถือ
