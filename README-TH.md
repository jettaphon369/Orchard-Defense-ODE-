# Orchard Defense ODE — Sprint 1

ฐาน Web App แบบโมดูลสำหรับ GitHub Pages

## มีแล้ว
- Three.js 0.180 แบบ ES Modules
- กล้องหมุนและซูม
- กระดาน 5 × 9
- Peachy / Sunny / Spud
- Grasshopper / Beetle
- ระบบแสงแดด
- ระบบเวฟ
- localStorage
- PWA เบื้องต้น
- โครงสร้างพร้อมเพิ่ม GLB, Animation, VFX และ Audio

## อัป GitHub Pages
อัปไฟล์และโฟลเดอร์ทั้งหมดโดยรักษาโครงสร้างเดิม แล้วเปิด:

Settings → Pages → Deploy from a branch → main → /root

## หมายเหตุสำหรับมือถือ
GitHub บนมือถือไม่สามารถแตก ZIP บนหน้าเว็บได้โดยตรง ต้องใช้แอปจัดการ Git เช่น Working Copy (iPhone/iPad), GitHub Codespaces หรืออัปผ่านคอมพิวเตอร์หนึ่งครั้ง
